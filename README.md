# AI Dynamic Story Engine

A Game Idea suggestor with plot and missions included.

## Tech Stack
- FastAPI (Python)
- React
- Ollama (Llama3 local model)

## Features
- Dynamic story generation
- World state management
- Branching choices
- Local LLM integration
- Fully free setup

## Run Backend

cd backend
venv\Scripts\activate
uvicorn app.main:app --reload

## Run Frontend

cd frontend
npm start
